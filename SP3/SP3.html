Sistema de ficheros y particiones

  • Tamaño del sector
  • Tamaño del bloque
  • Fragmentación interna
  • Fragmentación externa
  • Tipos de formateo:
    • Bajo nivel: Borra la totalidad del disco, sobrescribiendo todo con ceros e intenta reparar los sectores defectuosos. Requiere software específico del fabricante y no se realiza habitualmente desde el sistema operativo.
    • Medio nivel: Es un punto intermedio; no borra los archivos de datos, pero marca los sectores defectuosos encontrados (sin intentar repararlos). Permite la recuperación de archivos.
    • Alto nivel: No borra los archivos físicamente, sino que restablece el sistema de archivos (la tabla de asignación) e ignora los sectores defectuosos que pueda encontrar.

LDAP y SMB (Integración)

Para realizar la integración, instalamos los paquetes necesarios:

sudo apt install libnss-ldap libpam-ldap smbldap-tools -y

Editamos nano /etc/samba/smb.conf e incluimos lo siguiente dentro del bloque [global]:

# Configuració Backend LDAP
passdb backend = ldapsam:ldap://localhost
ldap suffix = dc=proves,dc=cat
ldap user suffix = ou=usuaris
ldap group suffix = ou=grups
# DN de l'administrador que has configurat abans
ldap admin dn = cn=admin,dc=proves,dc=cat
ldap ssl = no

Creamos el recurso compartido editando nuevamente el archivo /etc/samba/smb.conf y añadiendo:

[recurs_alu1]
comment = Carpeta privada alu1 
path = /srv/samba/alu1
browseable = yes
read only = no
guest ok = no
valid users = alu1
writable = yes
create mask = 0770
directory mask = 0770

Aplicamos los cambios creando el directorio, asignando permisos y reiniciando servicios:

mkdir -p /srv/samba/alu1
chmod 777 /srv/samba/alu1
systemctl restart smbd nmbd

Para verificar el funcionamiento:

  1. Desde el equipo cliente (asegurando conectividad previa), abrimos el navegador de archivos.
  2. Vamos a “Otras ubicaciones” e introducimos: smb://10.0.2.15/recurs_alu1.
  3. Se solicitará autenticación. Introducimos:
    • Usuario: alu1
    • Contraseña: (la definida en el archivo usu.ldif).
  4. Si la configuración es correcta, tendremos acceso para crear y editar carpetas.
