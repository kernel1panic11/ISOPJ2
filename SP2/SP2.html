Fase 1. Preparación del almacenamiento

Paso 1. Añadir disco virtual

Se añade un nuevo disco virtual de 20 GB tipo NVMe en VMware. Este disco se utilizará exclusivamente para prácticas de almacenamiento, particiones y control de cuotas, sin afectar al sistema principal.

Paso 2. Inicialización del disco

Desde el administrador de discos se inicializa el nuevo disco utilizando el esquema de partición GPT, adecuado para sistemas modernos y compatible con configuraciones actuales de Windows 11.

Paso 3. Creación de particiones

Se crean dos volúmenes diferenciados: uno principal en NTFS denominado Dades (15 GB), y otro secundario en FAT32 denominado Portable (5 GB). Esta separación permite trabajar con distintos sistemas de archivos según el uso.

Paso 4. Verificación con diskpart

Se utiliza diskpart para comprobar la correcta detección del disco y sus volúmenes, verificando tamaño, tipo y asignación de unidades.

Fase 2. Cuotas, usuarios y grupos

Paso 5. Configuración de cuotas

Se accede a las propiedades de la unidad Dades (NTFS) y se habilita la administración de cuotas, configurando un límite de 300 MB por usuario con advertencia previa.

Paso 6. Creación de usuarios

Se crean dos usuarios locales, alumne1 y alumne2, que serán utilizados para comprobar el comportamiento de las cuotas.

Paso 7. Creación de grupo

Se crea el grupo Limitats y se añaden ambos usuarios, permitiendo gestionar permisos de forma conjunta.

Paso 8. Prueba de cuotas

Se inicia sesión con alumne1 y se generan archivos en la unidad Dades hasta superar el límite. El sistema bloquea la escritura al alcanzar los 300 MB.
